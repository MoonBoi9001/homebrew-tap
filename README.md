# Homebrew Tap for apple-juice

Security-hardened macOS battery management CLI.

## Installation

```bash
brew tap MoonBoi9001/apple-juice
brew install apple-juice
```

## Post-install

Run the setup command:

```bash
sudo battery visudo $USER
```

Then start managing your battery:

```bash
battery maintain longevity
```
